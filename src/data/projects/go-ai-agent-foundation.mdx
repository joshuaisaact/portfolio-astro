A CLI agent in Go that talks to Claude and can use tools: read files, list directories, edit files, and run ripgrep searches. I built it to learn how tool-use agents work under the hood, parsing the model's tool call JSON, dispatching to the right handler, and feeding results back into the conversation loop.
