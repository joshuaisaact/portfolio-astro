A learning project: a CLI agent that uses Gemini to read files, write files, and run Python scripts in a loop until it finishes the task (or hits 20 iterations). All file operations are sandboxed to a working directory and script execution has a 30-second timeout.

I built it alongside the Go version to compare how the same agent pattern feels in both languages.
