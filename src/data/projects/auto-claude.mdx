Inspired by Karpathy's autoresearch but generalized beyond ML training. You define a problem in a `program.md` file -the editable source, a measurement command, and a metric to optimize -then the agent loops autonomously: reading the current state, proposing an edit, committing, measuring, and keeping or rolling back. Results so far include a 26% faster LRU cache than mnemonist, 3.7x faster glob matching than picomatch, and 53% faster JSON serialization than native `JSON.stringify`.
